# Картинки для сайта

Положи в эту папку (`public/`) файлы с такими именами — сайт подхватит их автоматически.

| Файл | Где используется |
|------|------------------|
| `hero-bg.jpg` | Фон главного экрана (Hero) |
| `service-1.jpg` | Услуга: Шлифовка срубов |
| `service-2.jpg` | Услуга: Консьержная шлифовка |
| `service-3.jpg` | Услуга: Покраска и пропитка |
| `portfolio-1.jpg` | Услуга: Антисептирование + портфолио |
| `portfolio-2.jpg` | Услуга: Ремонт швов + портфолио |
| `portfolio-3.jpg` | Услуга: Мытьё фасада + портфолио |
| `before.jpg` | Блок «До/После», проекты, портфолио |
| `after.jpg` | Блок «До/После», проекты, портфолио |
| `teply-shov-hero.jpg` | Фон первого экрана страницы «Теплый шов» (красивое фото теплого шва / герметизации) |

**Квиз** — создай папку `public/quiz/` и положи туда сгенерированные картинки:
| Файл | Вопрос квиза |
|------|--------------|
| `quiz-srub.jpg` | Сруб (бревенчатый) |
| `quiz-brus.jpg` | Дом из бруса |
| `quiz-kleyeny.jpg` | Клееный брус |
| `quiz-banja.jpg` | Баня/сауна |
| `quiz-shlifovka.jpg` | Шлифовка |
| `quiz-pokraska.jpg` | Покраска/пропитка |
| `quiz-antiseptik.jpg` | Антисептирование |
| `quiz-konopatka.jpg` | Конопатка швов |

Если файлов нет в `quiz/`, используются fallback-картинки из корня `public/`.

В референсе («Проекты с poehali» → app) этих файлов не было в архиве — только пути в коде. Если у тебя есть оригинальные картинки оттуда или свои фото работ, скопируй их в `public/` с указанными именами.
